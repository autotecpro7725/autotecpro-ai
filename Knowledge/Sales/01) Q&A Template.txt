Q: (Aftermarket Amplifier) Does the system support RCA output for amplifier? (For Chevy Silverado, Ford F150 (15+), Explorer, Mustang, Infiniti..etc)

A: 

Hi How are you doing? If your vehicle comes with an original system like OnStar, SYNC 2, or SYNC 3, you can still install an aftermarket amplifier or subwoofer. Simply just connect it to the existing original system. Since the original system remains in use, the audio signal must first pass through the original system before reaching the amplifier (Original/Aftermarket). If you connect the amplifier directly to the new system, you will bypass audio from the original system, such as the radio and other factory sources.

To summarize, our system retains the original setup and works alongside the factory radio, which handles the audio output. To connect an aftermarket amplifier, it must be wired to the back of the original radio. If connected directly to the new system, the amplifier will bypass the original radio's signal. 

Here's how the new system functions:

New screen → Original Radio (uses factory sound system) → Signal is sent to the original car amplifier & sound system.

Therefore, in order to connect an aftermarket amplifier, you can simply connect it to the original radio. This allows you to use the amplifier with both the factory radio and the new system simultaneously.

Q: Does the system support RCA output for amplifier? (For Audi, BMW, Mercedes-Benz..etc)

A: Since our system retains the original system (Original information system and radio). This is how the new system works: The signal goes from the New screen -> Original Radio (Use original sound system) -> The signal will be sent from the original radio to the original Car amplifier & Sound System. Therefore, in order to connect an aftermarket amplifier, you can connect directly to the original radio. Then you can use the new aftermarket amplifier on both the original radio and the new system at the same time.
