This system retains your original factory radio (such as OnStar, SYNC 2, or SYNC 3). Because of this, all audio still goes through the original radio.

Audio flow:

New screen → Original radio → Amplifier → Speakers

There is no RCA output from the new unit.
If you would like to install an aftermarket amplifier or subwoofer, it needs to be connected to the back of the original radio, not directly to the new screen.
If your amplifier already works with the factory system, it will continue to work the same way after installation.
If connected directly to the new screen, it may bypass the original radio and cause some factory audio (such as FM/AM) to not work.

Hi, how are you doing? I hope you’re doing well.

To clarify, this system retains your original factory radio (such as OnStar, SYNC 2, or SYNC 3), so all audio continues to go through the original system.

Audio flow:
New screen → Original radio → Amplifier → Speakers

There is no RCA output from the new unit. The aftermarket amplifier is not directly related to the new system, as it relies on the original radio for audio signal.

If you would like to install or use an aftermarket amplifier or subwoofer, it must be connected to the back of the original radio, not the new screen. If your amplifier already works with the factory system, it will continue to work the same way after installation.

Also, it is very important to confirm whether there is audio in Console mode (original system).

If there is no sound in Console mode, the issue is related to the original system or wiring.
If there is sound in Console mode, then the system is working correctly.

If the amplifier is connected directly to the new screen, it may bypass the original radio and cause some factory audio (such as FM/AM) to not work.
